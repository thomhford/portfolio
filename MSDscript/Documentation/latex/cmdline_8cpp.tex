\doxysubsection{Detailed Description}
Reads command line arguments and runs said commands. 

This program runs at the command line and can be given program arguments. For example,

./msdscript --help

This will run the program msdscript in the current directory with the argument --help. Arguments are passed to a program as the argc and argv arguments to main\+:

int \mbox{\hyperlink{main_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main(int argc, char$\ast$ argv\mbox{[}$\,$\mbox{]})}} \{ .... \}

The argc passed to main is always at least 1, because the first “argument” (at index 0) is the name of the program as it was run on the command line. If argc is 2, that means the program was given one argument, maybe as

./msdscript --help

and then argv\mbox{[}1\mbox{]} is that argument, which is \char`\"{}-\/-\/help\char`\"{} in this case.

Possible commands\+: --help --test
